## рҹ“қ ADLC Phases

| Phase | Activities | Deliverables |
|:---|:---|:---|
| **Design** | Requirements, architecture, security review | Design docs, ADRs |
| **Develop** | Implementation, code review, documentation | Working code, tests |
| **Test** | Unit, integration, evaluation, load testing | Test reports |
| **Deploy** | Staging, production, rollback planning | Deployed system |
| **Monitor** | Observability, alerting, performance tracking | Dashboards |
| **Iterate** | Feedback analysis, improvement planning | Backlog items |
