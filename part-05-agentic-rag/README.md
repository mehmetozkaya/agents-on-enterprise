# Part 05 - Agentic RAG

This section focuses on building agentic RAG (Retrieval-Augmented Generation) systems, which combine the capabilities of AI agents with retrieval mechanisms to enhance their performance and versatility. You'll learn how to create agents that can interact with external data sources, retrieve relevant information, and generate responses based on that information.
