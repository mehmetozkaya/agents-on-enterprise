## 🎯 What You'll Learn

This chapter provides a comprehensive survey of AI agent frameworks, with a focus on the .NET ecosystem.

### Topics Covered

- **Microsoft Agent Framework (MAF)** — Microsoft's official framework for building AI agents
- **Semantic Kernel** — Microsoft's SDK for AI orchestration
- **AutoGen** — Multi-agent conversation framework
- **LangChain (.NET)** — Popular agent framework ecosystem
- **Framework Selection Criteria** — Choosing the right tool for your enterprise

---

## 🔍 Framework Comparison

| Framework | Strengths | Best For |
|:---|:---|:---|
| **Microsoft Agent Framework** | Native .NET, enterprise-ready, Azure integration | Production enterprise systems |
| **Semantic Kernel** | Flexible orchestration, plugin architecture | AI orchestration scenarios |
| **AutoGen** | Multi-agent patterns, research-driven | Complex multi-agent systems |
| **LangChain** | Large ecosystem, many integrations | Rapid prototyping |

---

## 💡 Key Insight

> **This book focuses on the Microsoft Agent Framework (MAF)** — it's purpose-built for enterprise .NET applications with first-class Azure integration.

MAF provides:
- Native C# development experience
- Built-in enterprise patterns
- Azure OpenAI and Azure AI Foundry integration
- Production-ready observability
